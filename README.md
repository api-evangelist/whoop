# WHOOP (whoop)

WHOOP is a fitness and recovery wearable platform providing a REST API for accessing sleep, recovery, strain, and workout metrics from the WHOOP band via OAuth2. The WHOOP Developer Platform allows developers to build applications that access member health and performance data including physiological cycles, sleep stages, recovery scores, heart rate, body measurements, and workout records.
